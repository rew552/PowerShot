# >_ PowerShot v2.0

クリップボードの画像を検知し、即座にリネーム・管理・保存ができる生産性向上ツールです。<br>
インストール不要で、batファイルをダブルクリックするだけで起動します。<br>
コードはbat、PowerShell、C#（PowerShell上で実行）、xamlで構成されています。

---

## 🚀 主な機能

### 1. キャプチャ機能
- クリップボードに画像がコピーされると、自動的に保存用UIが前面に表示されます。
- 作業を止めることなく、連続的なスクリーンショットの保存操作が可能です。

### 2. ファイル自動命名機能
- **接頭辞**: 自由な文字列を入力可能。
- **連番**: 選択した桁数の連番を自動生成。
- **タイムスタンプ**: 「日時」を選択すると、現在の時刻に基づいた一意のファイル名を生成します。
- **設定の保存**: 最後に使用した拡張子や連番設定を記憶します。（ツールを閉じると設定はリセットされます）

### 3. システム情報の埋め込み
- 「💻 システム情報を画像に埋め込む」をオンにすると、画像左下に **ホスト名 / ユーザ名 / IPアドレス /** を自動で合成します。エビデンス作成に最適です。

### 4. 統合ファイルエクスプローラー
- 保存先フォルダ内のファイルをリアルタイムに一覧表示。
- アプリ内から直接フォルダを作成、削除できます。
- ファイルをダブルクリックすると、画像プレビューが表示されます。
- フォルダをダブルクリックすると、そのフォルダに移動します。
- 戻るボタンで、一つ上の階層に戻れます。

---

## 🛠️ 使い方

1.  `powershot.bat` を実行して起動します。
2.  Windows標準機能（`Win + Shift + S` など）でスクリーンショットを撮るか、画像をコピーします。
3.  PowerShotの画面が自動で表示されます。
4.  ファイル名を設定し、**保存 (Enter)** を押します。
5.  右側のエクスプローラーで、保存されたファイルをプレビュー・管理できます。
> 保存先フォルダは、スクリプト実行時に作成される「Screenshots」フォルダです。

---

## ⌨️ ショートカットキー

| キー | 動作 |
| :--- | :--- |
| **Enter** | 設定した名前で画像を保存 |
| **Delete** | エクスプローラーで選択中のファイルを削除 |
| **Esc** | スクリーンショット保存画面を閉じる（クリップボード監視は継続） |
