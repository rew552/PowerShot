# >\_ PowerShot v3.0

クリップボードの画像を検知し、即座に編集・管理・保存ができる生産性向上ツールです。<br>
インストール不要で、batファイルをダブルクリックするだけで起動します。<br>
外部ライブラリを一切使わず、Windows標準のPowerShellとC# (WPF) だけで構成されています。

---

## 🚀 主な機能

### 1. キャプチャ＆自動ポップアップ

- クリップボードに画像がコピーされると、自動的に保存用UIが前面に表示されます。
- 作業を止めることなく、連続的なスクリーンショットの保存操作が可能です。

### 2. 高度な画像編集機能

- **トリミング (切り抜き)**: プレビュー画面上でドラッグ操作し、直感的に必要な部分だけを切り出すことができます。
- **オーバーレイ (テキスト合成)**: 画像の四隅に任意のテキストやシステム情報（ホスト名 / ユーザ名 / IPアドレス）、撮影日時を透かしとして合成可能。エビデンス作成に最適です。

### 3. スマートなファイル命名

- **接頭辞**: 自由な文字列を入力可能。
- **連番・日時**: 選択した桁数の連番、またはカスタマイズ可能なタイムスタンプ（例: `yyyyMMdd-HHmmss`）を自動生成。
- **フォーマット選択**: `.jpg` (画質指定可能) と `.png` (無劣化) から選択できます。

### 4. 統合ファイルエクスプローラー

- 保存先フォルダ内のファイルをリアルタイムに一覧表示。
- アプリ内から直接フォルダを作成、削除できます。
- ファイルをダブルクリックすると、画像プレビューが表示されます。

### 5. ⚙️ 設定カスタマイズ

- アプリ右上の歯車アイコンから設定画面を開き、各種デフォルト動作をカスタマイズ可能。
- 保存先フォルダの変更、タイムスタンプ形式の変更、デフォルト拡張子の設定などが行え、`settings.json` に自動保存されます。

### 6. 独立型 画像ビューア (`powershot-viewer.bat`)

- PowerShot本体とは独立してブラウザで起動する、軽量・高速な画像ビューアです。
- フォルダツリーでのナビゲーション、ファイル名検索、ソート機能（名前順・日時順など）を搭載。
- サムネイルをクリックすると拡大表示になり、**← / → キー** で画像を閲覧できます。

---

## 🛠️ 使い方

1.  `powershot.bat` を実行して常駐させます。
2.  Windows標準機能（`Win + Shift + S` など）でスクリーンショットを撮るか、画像をコピーします。
3.  PowerShotの画面が自動で表示されます。必要に応じてトリミングやテキスト合成を設定します。
4.  ファイル名を確認し、**保存 (Enter)** を押します。
5.  保存した画像群を一覧で見たい場合は、同じディレクトリにある `powershot-viewer.bat` を実行します。

> デフォルトの保存先フォルダは、スクリプト実行ディレクトリ内の「Screenshots」フォルダです。

---

## ⌨️ ショートカットキー

| キー       | 動作                                                           |
| :--------- | :------------------------------------------------------------- |
| **Enter**  | 設定した名前で画像を保存                                       |
| **Delete** | エクスプローラーで選択中のファイルを削除                       |
| **Esc**    | スクリーンショット保存画面を閉じる（クリップボード監視は継続） |
